# K线学习交流平台

一个功能完善、用户友好的K线学习交流平台，旨在帮助投资者学习K线分析技巧、了解K线形态、掌握技术指标，提升投资决策能力。

## 功能特性

### 核心功能
- **K线技巧展示**：包含基础K线形态、组合K线形态、技术指标、实战技巧等分类
- **K线四字成语**：详细讲解看涨、看跌、盘整/变盘、特殊强势/弱势形态等四大类K线成语
- **技能卡片交互**：点击技能卡片弹出详细介绍，包含视频讲解位置
- **响应式设计**：自适应手机和电脑设备，确保在不同屏幕尺寸下都能正常显示
- **主题切换**：支持白天/夜晚模式切换，根据用户偏好自动保存
- **折叠/展开功能**：分类内容可折叠/展开，方便浏览
- **返回顶部按钮**：快速返回页面顶部
- **平滑滚动**：锚点导航平滑滚动效果

### 技术特性
- **语义化HTML**：使用HTML5语义化标签，提升页面结构清晰度和可访问性
- **CSS变量**：使用CSS变量实现主题切换，代码更简洁易维护
- **Flexbox和Grid布局**：实现灵活的页面布局
- **图片优化**：所有图片都添加了loading、width、height和alt属性，提升加载性能
- **无障碍支持**：添加ARIA属性，提升页面可访问性
- **SEO优化**：添加meta标签、结构化数据，提升搜索引擎优化效果
- **交互动画**：添加模态框显示/隐藏动画，提升用户体验
- **容错处理**：添加代码容错处理，确保页面在各种情况下都能正常运行

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **布局**：Flexbox, Grid
- **响应式设计**：Media Queries
- **主题切换**：CSS Variables, localStorage
- **图标**：通过API动态生成

## 如何使用

1. **直接打开**：在浏览器中直接打开 `index.html` 文件
2. **本地服务器**：使用本地服务器（如Live Server）打开项目

### 导航操作
- 点击导航栏链接跳转到对应 section
- 在移动设备上点击汉堡菜单按钮展开/收起导航菜单

### 主题切换
- 点击导航栏右侧的太阳/月亮图标切换白天/夜晚模式
- 主题偏好会自动保存到本地存储

### 技能卡片交互
- 点击任意技能卡片查看详细介绍
- 在弹出的模态框中可以看到技能的详细说明和视频讲解位置
- 点击模态框外部或关闭按钮关闭模态框

### 分类展开/收起
- 点击分类标题右侧的「展开」/「收起」按钮展开/收起分类内容
